# 测试功能（此栏目以下文件暂时放置后续修改）

下列正则需要在vscode中使用，爬虫copy下页面后，用来处理下列展示

> 这个正则表达式的含义是：匹配以一位数字或两位数字开头的行，或者匹配至少3行连续的空白行。在VS Code的搜索框中输入这个正则表达式，然后点击搜索按钮，就可以找到同时满足这两个条件的行了。

```js
  ^\d{1,2}.+$\n|^\s*$\n(\s*$\n){2,}
  // 解决下列展示占位
  1  
  2  
  3  
  4  
  5  
  6  
  7  
  8  
  9  
```

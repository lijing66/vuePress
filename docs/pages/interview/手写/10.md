# 统计数组中元素出现的频率

## 需求

\["b", "c","b", "c","a", "b", "c"\] 统计每个元素出现的次数,结果是： {a：1，b: 3, c: 3}

```js
const arr = ["b", "c","b", "c","a", "b", "c"] 

const obj = {}
// 你的代码 ...

console.log(obj) // ==> {a：1， b: 3, c:  3}
```

# reduce

## 简版

+   reduce 作用：给每一个元素执行一次回调函数，并且返回最终的结果
    
+   reduce 参数有两个：第一个是回调函数，第二个是初始值。 一般初始值都会设置为 0，目的是避免空数组报错
    
    +   回调函数内部常用的形参有三个，依次是累加值、当前元素、当前下标
    +   回调函数内部的 return 返回值就是下一次回调的累加值
+   reduce 方法的返回值:就是执行完遍历之后，最终的累加值

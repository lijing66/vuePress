# 块级元素和行内元素和行内块的区别

通过设置 display 属性可以改变元素的类型。

## 块级元素(block)的特点

1.  占据一行
2.  高度，行高以及外边距和内边距都可设置
3.  宽度默认为是它的容器的100%，但可以设定一个宽度
4.  它可以容纳内联元素和其他块元素

常见的有：div, h1, p

## inline元素的特点

1.  和其他元素共享一行上
2.  高，行高及外边距和内边距不可改变
3.  宽度就是它的文字或图片的宽度，不可改变

常见的有：i span a

## inline-block元素的特点

1.  和其他元素都在一行上
2.  行高及外边距和内边距都可改变；

常见的有：buttton img, input
